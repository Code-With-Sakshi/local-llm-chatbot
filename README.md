# 🤖 Local LLM AI Chatbot

An AI-powered chatbot that runs **completely offline** using a locally hosted Large Language Model.  
This project uses **Phi-3 via Ollama** and a **Gradio-based web interface** to create a simple and interactive chatbot.



## Features
🧠 Runs on a **local Large Language Model (Phi-3)**
🌐 **No internet required**
💻 Simple **web interface using Gradio**
⚡ Local API communication with **Ollama**
🎨 Custom UI styling for better user experience
🗑 Clear conversation functionality


## 🛠 Tech Stack

**Python**
**Gradio**
**Ollama**
**Phi-3 Model**
**Requests Library**
